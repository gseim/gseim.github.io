xbe Documents¶

In this section, documents for all xbe’s are presented. These documents can also be accessed from the GSEIM schematic capture GUI as follows.

  • From the library (right-hand menu in the GUI), select the xbe of interest by left-clicking. Then right-click to view the document.
  • To view the document on a specific xbe in a circuit scematic, right-click on that element and then select Show Document.

Sub-circuit Documents¶

Sub-circuits (hierarchical blocks) are made up of basic blocks and can also include other sub-circuits. In this section, documents for sub-circuits already available with the GSEIM distribution are presented. These documents can also be accessed from the GSEIM schematic capture GUI as follows.

  • From the block tree panel (right-hand menu in the GUI), select the sub-circuit of interest by left-clicking. Then right-click to view the document.
  • To view the document for a specific sub-circuit in a circuit schematic, right-click on that sub-circuit and then select Show Document.
